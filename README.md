## 프로젝트

### 오픈소스 라이브러리 & 도구

- **[combinatorial-config](https://github.com/devcomfort/combinatorial-config)** - 실험 설정의 모든 조합을 자동으로 생성하는 Python 라이브러리. Hydra 등 설정 관리 도구와 함께 체계적인 하이퍼파라미터 탐색에 유용합니다.

- **[file_type_detector](https://github.com/devcomfort/file_type_detector)** - 파일 내용 분석과 파일 시그니처 기반으로 파일 형식을 감지하는 Python 라이브러리.

- **[devcomfort-text-transcoder](https://github.com/devcomfort/devcomfort-text-transcoder)** - Node.js와 브라우저 환경에서 텍스트 인코딩을 변환하는 경량 라이브러리. UTF-8, EUC-KR, EUC-JP 등 다양한 인코딩을 지원합니다.

- **[is-html-element](https://github.com/devcomfort/is-html-element)** - HTML 엘리먼트 타입 검사 및 유효성 검증 유틸리티 라이브러리.

- **[ubuntu-disk-toolkit](https://github.com/devcomfort/ubuntu-disk-toolkit)** - Ubuntu 디스크 관리 및 작업을 위한 CLI 툴킷 (개발 중).
